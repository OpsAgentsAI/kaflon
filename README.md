# Kaflon · כפלון

> Free, open-source multiplication & division practice for kids. Built with [Claude Code](https://claude.com/claude-code) in an evening.

**Live:** https://opsagentsai.github.io/kaflon

A single HTML file. No accounts, no ads, no tracking, no IAPs. Works on any phone, tablet, or laptop. Hebrew (RTL) and English. Fork it, tweak the number range, ship it to your kid.

## Features (v0)

- × / ÷ / mixed modes
- Single-table drill (1–10) or random across the full set
- Up to 10 × 10 = 100 (whole-number division only)
- Score + streak counter
- Hebrew (RTL) + English toggle
- Mobile-first, big tap targets, kid-friendly colors

## Why

I built this for my son in one evening, with Claude Code. Every paid kids-math app I tried wanted an account, a subscription, or sneaked an ad in. He just needed something to drill on. So here it is, for him and for yours.

If you want to adapt it — change the range, swap the colors, add your kid's name as the high-score holder — fork the file, edit, deploy. It's one HTML file. View source is the documentation.

## Local use

Open `index.html` in any browser. That's it.

## Fork & deploy

```bash
gh repo fork OpsAgentsAI/kaflon
# edit index.html as you like
git push
# enable GitHub Pages on your fork → live URL in 30 seconds
```
